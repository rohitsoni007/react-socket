# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2025-12-16

### Added
- Initial release of react-socket-client
- React.js wrapper for socket.io-client
- SocketProvider component for easy Socket.IO integration
- Socket component for handling socket events
- Utility hooks: useSocket, useSocketEmit, useSocketListener, useSocketStatus
- Configurable logging with custom log levels
- TypeScript support with generic types for event data
- Built-in error handling
- Automatic connection management
- Connection status tracking (connecting, connected, disconnected, error)
